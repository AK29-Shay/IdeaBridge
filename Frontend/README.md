# Frontend — Project Search UI

Simple static frontend that calls the backend search API at `http://localhost:3000/api/projects/search`.

This folder now contains a small React + Vite frontend (JavaScript) that calls the backend search API at `http://localhost:3000/api/projects/search`.

Run (install deps then start dev server):

```powershell
cd Frontend
npm install
npm run dev
```

Open http://localhost:5173 (Vite default) and use the search UI. Ensure the backend is running on http://localhost:3000.
